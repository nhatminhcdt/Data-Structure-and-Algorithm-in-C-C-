# Data Structure and Algorithms

## Environment
- OS: Ubuntu 20.04 LTS (WSL2)
- IDE: Visual Studio Code
- Python: 3.8.10
- Compiler: gcc 9.4.0
- CMake: 3.16.3


## C++ Build
At `/leetcode/cpp/` of the project, run the following commands:
```bash
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ctest -V
```
